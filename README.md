**API - Planejador de Caminhos**



**Descrição:** API desenvolvida na disciplina de Desenvolvimento Web Backend para o cálculo e exposição do caminho mais curto entre localidades específicas do estado do Rio Grande do Norte, também mostrando as rotas e pontos geográficos percorridos nesse caminho, basicamente a simulação de um GPS. Esse caminho mais curto é, de fato, o caminho de menor custo em grafo calculado utilizando o algoritmo A\*. As rotas e nós, assim como a idéia geral, foram disponibilizadas pelo professor Adelardo Medeiros de Programação Avançada.



**Ferramentas:** foi feita a implementação de rotas RESTful, a manipulação de arquivos JSON como banco de dados, e a aplicaçao de segurança básica com hashing de dados. Mais especificamente, as dependencias usadas até o desenvolvimento final foram: express, jsonwebtoken, bcrypt, express-validator, lowdb, dotenv, pug, mongodb, mongoose, e pug.
